# React + Vite + TailWindCSS - A Template Repo

It is a template repository for you to start a project using React, TailwindCss, Vite, Alias Paths, React-Router-Dom and FSD metodology . You just need to create a new repository from this template repo by clicking the button at the top right corner of this page.
